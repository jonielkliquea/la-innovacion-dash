Web App Innova Centro

El desarrollo de esta webapp surge de la necesidad de abordar las limitaciones y riesgos asociados con la práctica de hardcodear código directamente en un sitio web en producción. Esta estrategia no solo restringe la capacidad de modificar el sitio a quienes poseen un conocimiento profundo del código, sino que también dificulta la implementación de cambios rápidos y seguros, lo que puede afectar negativamente la eficiencia operativa y la capacidad de responder a nuevas necesidades.

La solución propuesta ha sido la creación de un conjunto de módulos que automatizan tareas específicas del sitio web de Innovacentro. Estos módulos permiten que el equipo de la empresa pueda gestionar elementos clave del frontend sin necesidad de intervención directa en el código fuente, lo que agiliza los procesos, mejora la seguridad y minimiza los errores. Todos los módulos están integrados en un Dashboard central y están conectados a una API que sincroniza el backoffice con la versión pública del sitio.

Introducción a la Documentación del Conjunto de Webapp para Innovacentro

El Dashboard es la página principal que proporciona una visión general del estado de los módulos implementados. Aquí se muestra qué módulos están activos, las fechas de vigencia y métricas importantes. Además, incluye accesos rápidos a los demás módulos a través de íconos.

Módulos Desarrollados

  • Dashboard: Configura bancos, porcentajes de devolución, estados, fechas, y límites de devolución para mostrarse en el frontend.
  • Módulo de CashBack: Vista cliente.
  • Módulo de Nuevos: Asigna ID a artículos nuevos, marcándolos como "Nuevo".
  • Módulo de Ícono de Oferta: Gestiona stickers de oferta y fechas de activación.
  • Módulo de Artículos en Tienda Física: Lista artículos solo disponibles en tienda física, sin opción de compra online.

Seguridad y Acceso

El acceso a esta webapp está restringido y se realiza únicamente a través de un correo electrónico corporativo de Google y un usuario autorizado. La conexión entre el backoffice y la versión pública del sitio web está asegurada a través de una API privada que sincroniza los datos de manera eficiente.
