Uso de la Web App Innova Centro

Este sitio web y su contenido han sido desarrollados para uso exclusivo del equipo de Innova Centro. Está prohibido el acceso no autorizado a cualquier parte del sistema, incluyendo el Dashboard y los módulos que automatizan las tareas del sitio web.

El uso de esta aplicación web está limitado a usuarios autorizados que cuentan con un correo electrónico corporativo de Google y han sido previamente registrados en el sistema. Cualquier intento de acceso no autorizado será considerado como una violación de las políticas de seguridad de Innova Centro.

Los datos manejados por esta webapp están protegidos a través de una API privada que asegura la sincronización entre el backoffice y la versión pública del sitio web. El acceso, uso o divulgación indebida de esta información es una infracción a las políticas de seguridad y confidencialidad de Innova Centro.
